Multimodal UI Analyzer

Automatic setup for local UI screenshot analysis with Ollama and Semantic Kernel.

Open app Ollama page
1

Run Auto Installer

Copy one command for your operating system. It downloads the app, prepares Ollama, starts the local web server, waits for it to respond, and opens the browser.

Windows PowerShell
macOS Apple Silicon
macOS Intel
2

What The Installer Does

Windows: downloads the full ZIP package into %LOCALAPPDATA%\Programs\MultimodalUIAnalyzer, checks Ollama, and installs it with irm https://ollama.com/install.ps1 | iex when needed.

macOS: checks ollama, installs it with brew install ollama when Homebrew is available, downloads the app ZIP, removes quarantine attributes, and starts the server.

3

Manual Package Fallback

You can also download the package manually. The launcher inside the ZIP prepares Ollama, starts the server, and opens the browser.

Windows x64 ZIP Full package with the app and web UI macOS Apple Silicon M1, M2, M3, M4 macOS Intel Older Intel-based Macs
4

Open App And Analyze

Once the server is running, open the app, choose a vision model, upload a screenshot, and click Analyze. If the selected Ollama model is missing, the app downloads only that model and shows progress.
