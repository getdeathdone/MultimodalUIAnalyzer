Multimodal UI Analyzer

Автоматическая настройка локального анализа UI-скриншотов через Ollama и Semantic Kernel.

Open app Ollama page
1

Run auto installer

Скопируйте одну команду под свою ОС. Она скачает приложение, подготовит Ollama и запустит локальный веб-сервер. Браузер не может сделать это сам без вашего действия, поэтому команда запускается в PowerShell или Terminal.

Windows PowerShell
macOS Apple Silicon
macOS Intel
2

What installer does

Windows: скачивает EXE в %LOCALAPPDATA%\Programs\MultimodalUIAnalyzer, проверяет наличие Ollama, при необходимости скачивает OllamaSetup.exe и запускает его.

macOS: проверяет ollama, при наличии Homebrew выполняет brew install ollama, скачивает ZIP приложения, снимает quarantine-атрибуты и запускает сервер.

3

Manual package fallback

Если хотите без команды, можно скачать пакет вручную. Launcher внутри ZIP тоже теперь пытается подготовить Ollama, а затем поднимает сервер и открывает браузер.

Windows x64 EXE Самый простой вариант для Windows macOS Apple Silicon M1, M2, M3, M4 macOS Intel Старые Mac на Intel
4

Open app and analyze

Когда сервер запущен, откройте приложение, выберите vision-модель, загрузите скриншот и нажмите Analyze. Если выбранной модели ещё нет локально, приложение скачает только её и покажет прогресс.
