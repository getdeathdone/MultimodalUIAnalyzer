What this app does

Analyze UI screenshots into structured JSON

Multimodal UI Analyzer is a local web app for uploading product or mobile UI screenshots, sending them to a vision model, and getting a clean JSON response with detected elements, approximate bounds, hierarchy, and accessibility notes.

  • Works with local Ollama vision models or cloud providers through Semantic Kernel.
  • Downloads only the model you choose, with progress shown inside the app.
  • Runs as a local web server at localhost:5088.

Install The Local Package

A browser page cannot install or launch a local program by itself. The easiest path is to copy one command for your operating system. It downloads the app, prepares Ollama, and opens the web UI.

Windows: One PowerShell Command

The command downloads the full ZIP package into %LOCALAPPDATA%\Programs\MultimodalUIAnalyzer, installs Ollama with the fast PowerShell install script when needed, and starts the app.

Windows PowerShell

macOS: One Terminal Command

The command installs Ollama with Homebrew, downloads the app archive, removes quarantine attributes, and starts the local server. If Homebrew is not installed, the script opens the Ollama download page and stops with a clear message.
