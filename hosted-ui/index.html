Установите локальный пакет

Браузер не может сам установить и запустить локальную программу без вашего действия. Самый простой вариант: скопировать одну команду для вашей ОС. Она скачает приложение, подготовит Ollama и откроет веб-интерфейс.

Windows: одна команда в PowerShell

Команда скачает EXE в %LOCALAPPDATA%\Programs\MultimodalUIAnalyzer, при необходимости поставит Ollama через быстрый PowerShell install script и запустит приложение.

Windows PowerShell

macOS: одна команда в Terminal

Команда поставит Ollama через Homebrew, скачает архив приложения, снимет quarantine-атрибуты и запустит локальный сервер. Если Homebrew не установлен, скрипт откроет страницу установки Ollama и остановится с понятным сообщением.
